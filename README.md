## 一期规则说明
### 通过古诗词转换成密码
#### 必备规则
1.两句连在一起，取拼音首字母，句首字母大写；  

2.数字规则
- 谐音líng/lín换成0、零；形象字无、女、母、雌、阴；  
- 谐音yī换成1；形象字有、男、公、雄、阳、单、独；
- ai/两换成2，亦可替换双、对
- 谐音san/shan换成3
- 谐音si/shi换成4
- 谐音wu/wo换成5，可替换我、舞
- 谐音liu/lu/lü换成6
- 谐音qi/qu/chi换成7，
- 谐音ba/bai换成8，
- 谐音jiu换成9，  

3.补位规则  
常用的包括往前补位、往后补位等  
先给出统一的补位方案：
不足八位的，往后依次添加wabjtam(我爱北京天安门)，补足八位。  
为什么一共就七个字？对输入肯定判空啊 :)
